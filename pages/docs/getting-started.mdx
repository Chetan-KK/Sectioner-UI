To get started with Sectioner UI, follow these steps:

1. Create a [React](docs/setup/react) or [Next.js](docs/setup/next) project.
2. Copy the CSS variables into your `globals.css` file.

```css showLineNumbers filename="globals.css" copy
/* Importing Poppins font */
@import url("https://fonts.googleapis.com/css2?family=Poppins:wght@100;200;300;400;500;600;700;800;900&display=swap");

/* Resetting default margin and padding */
*,
*::before,
*::after {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* CSS variables used in components */
:root {
  --background: 255 100% 100%;
  --primary: 200 100% 50%;
  --primary-foreground: 218 100% 50%;
  --secondary: 319 100% 50%;
  --secondary-foreground: 277 100% 50%;
  --text: 255 100% 0%;
  --text-foreground: 255 100% 100%;
  --sub-text: 255 0% 30%;
  font-family: "Poppins", sans-serif;
}

/* CSS variables for dark mode */
.dark {
  --background: 255 100% 0%;
  --primary: 200 100% 50%;
  --primary-foreground: 218 100% 50%;
  --secondary: 319 100% 50%;
  --secondary-foreground: 277 100% 50%;
  --text: 255 100% 100%;
  --text-foreground: 255 100% 0%;
  --sub-text: 255 0% 50%;
}

body {
  background-color: hsl(var(--background));
  color: hsl(var(--text));
}
```
