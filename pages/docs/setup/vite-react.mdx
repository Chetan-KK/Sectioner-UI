# React with Vite Setup
### steps

import { Tabs } from "nextra/components";

1. Open your terminal.
2. Run the following command to create a new React app:

<Tabs items={["npm", "yarn", "pnpm", "bun"]}>
  <Tabs.Tab>
  ```bash copy
  npm create vite@latest
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash copy
  yarn create vite
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash copy
  pnpm create vite
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash copy
  bun create vite
  ```
  </Tabs.Tab>
</Tabs>

3. Enter your Project name in prompt
4. Select `React` and then `javascript`
5. cd in `your-project-name`
6. install packages
7. copy these css into `index.css`


**note**: create separate folder for all sections/components in `sectioner` folder.
