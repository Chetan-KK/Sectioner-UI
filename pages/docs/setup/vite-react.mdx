### Add CSS Variables

Copy and replace the following CSS code into your `index.css` file to apply global variables to your project:

```css showLineNumbers filename="index.css" copy
/* importing poppins font */
@import url("https://fonts.googleapis.com/css2?family=Poppins:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap");

/* removing extra margin and padding */
*,
*::before,
*::after {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* css variables used in components */
:root {
  --primary: 200 100% 50%;
  --primary-forground: 218 100% 50%;
  --secondary: 319 100% 50%;
  --secondary-forground: 277 100% 50%;
  --text: 255 100% 0%;
  --text-forground: 255 100% 100%;
  --sub-text: 255 0% 30%;
  font-family: "Poppins", Tahoma, Geneva, Verdana, sans-serif;
}

/* css variables for dark mode */
.dark {
  --primary: 200 100% 50%;
  --primary-forground: 218 100% 50%;
  --secondary: 319 100% 50%;
  --secondary-forground: 277 100% 50%;
  --text: 255 100% 100%;
  --text-forground: 255 100% 0%;
  --sub-text: 255 0% 30%;
}
```
This code defines global color variables and imports the **'Poppins'** font for consistent styling across your application.

### Create the Sectioner Components Folder

Create the `sectioner` folder under `src/components` in your project.

**Alternatively:**

Open your project directory and run the following command to create the `sectioner` folder:

```bash copy
  mkdir -p src/components/sectioner
```

- All your Sectioner components will reside here.
- You can change the folder structure if you prefer.
